# Story: Refine Data Upload Wording

## User Story
As a Case Editor, I want to see clearer labels on the data upload interface so that I know exactly which template to download and what type of file to upload.

## Acceptance Criteria
- [ ] The "Download template" button text is changed to "Download required data template".
- [ ] The "Upload your data" header text is changed to "Upload your validated data template".
- [ ] The UI layout remains clean and functional on 1280x720 screens.
- [ ] The download and upload functionality remains fully operational.

## Technical Notes
- Target file: `frontend/src/pages/cases/components/CaseForm.js`
- Components: Ant Design `Button`, `Tabs`, `Dragger`.

# UI QA Guide - Issue #746

This guide outlines the steps to verify the UI refinements and description clarifications for the Step 3 visualizations.

## Role Identification
- **Roles Affected**: All users with access to view Case results (Admin, Internal, External Advanced, External Regular).

## Happy Path: Verification of Clarifications

1. **Action**: Open any existing Case or create a new one.
2. **Action**: Navigate to **Step 3 (Understand Income Gap)**.
3. **Action**: Locate the **"Needed Income Level"** chart.
4. **Check**: Verify the description text now includes: *"The total additional income needed is allocated proportionally based on the current share of each income source in total household income."*
5. **Action**: Locate the **"Household Income Composition"** chart.
6. **Check**: Verify the description text is clear and the container height aligns with the neighboring chart.

## Visual Alignment Check

1. **Action**: Resize the browser window to approximately 1366x768.
2. **Check**: Ensure that the description boxes for both charts in Step 3 have consistent heights and do not overlap with other UI elements.
3. **Check**: Ensure there is no vertical text clipping.
